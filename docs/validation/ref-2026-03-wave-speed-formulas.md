| 実装 | 基準書式番号 | 基準書記述 | 一致度 |

### K₁の定義について

基準書のPDF抽出テキストは数式レイアウトが崩れており、K₁の係数（1 または 2）を明確に読み取れなかった。
以下の計算が式(8.3.7)と一致することを確認した：

```
K₁ = L·V / (g·H₀·tν)
Hmax/H₀ = K₁/2 + √((K₁/2)² + 1)
         = (K₁ + √(K₁² + 4)) / 2   … 実装形式
```

| 強化プラスチック複合管（FW）| FW 5〜1種: 21.6, 19.6, 16.7, 15.2, 14.7 | 51（grp_fw 単一値） | ⚠ 不一致の可能性 |

### 問題の詳細

**技術書 表-8.2.1 の注記**:
> 注2）FW成形の5〜1種管の値を示す。管級や用途により本表以外の値の場合もある。

この注記と数値（21.6, 19.6, 16.7, 15.2, 14.7）から、これらは **強化プラスチック複合管（FW成形）** の各種（5〜1種）に対応する値と判断される。

- `grp_fw = 51e6` → 技術書に該当値なし。**誤値の疑いが強い。**
- `wdpe1〜5` の値（21.6〜14.7）→ GRP-FW 管の値が誤って割り当てられている可能性。水道配水用ポリエチレン管の正しい Eₛ はPDFから読み取れなかった。

**対応方針**:
- GRP-FW (`grp_fw`) の種別を 1〜5種に分割するか、代表値（例: 3種 = 16.7）を使用するか検討
- 水道配水用ポリエチレン管 (`wdpe`) の Eₛ を別途確認（JIS K 6762 等）

## 3. 数値検証（デモケースによる手計算との照合）

### ケース01: ダクタイル鋳鉄管 φ300mm 急閉そく

**入力**: D=0.300m, t=0.007m, L=500m, Eₛ=160×10⁶ kN/m², C₁=1.0, V₀=1.0m/s, H₀=30m, tν=0.5s

**手計算**:
```
分母 = (9.8/9.8) × (1/2.03×10⁶ + 0.300×1.0/(160×10⁶×0.007))
     = 1.0 × (4.926×10⁻⁷ + 2.679×10⁻⁷)
     = 7.605×10⁻⁷
a = 1/√(7.605×10⁻⁷) = 1/8.721×10⁻⁴ = 1146.7 m/s
```

**実装結果**: a = 1146.7 m/s ✓

**水撃圧**:
```
2L/a = 2×500/1146.7 = 0.872 s
tν=0.5s < 0.872s → 急閉そく
ΔH = a/g × V₀ = 1146.7/9.8 × 1.0 = 117.0 m
設計水圧 = (30+117.0)×9.8/1000 = 1.441 MPa
```

**実装結果**: ΔH = 117.0m, 設計水圧 = 1.441 MPa ✓

### ケース02: 同管 緩閉そく（tν=10s）

```
K₁ = 500×1.0/(9.8×30×10) = 0.1701
Hmax = 30/2 × (0.1701 + √(0.1701²+4)) = 15 × 2.177 = 32.66 m
設計水圧 = 32.66×9.8/1000 = 0.320 MPa
```

**実装結果**: Hmax = 32.66m, 設計水圧 = 0.320 MPa ✓
